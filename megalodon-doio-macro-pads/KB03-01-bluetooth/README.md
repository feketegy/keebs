# Megalodon DOIO KB03B-01 Bluetooth

## Notes

- Only works with its own USB-A to USB-C cable.
- The VIA keymap JSON file is custom, to map the buttons correctly to this exact model.
- Need to use VIA 1.3.1 to map the keys.
